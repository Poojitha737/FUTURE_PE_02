# Prompt: Hook Generator

## Purpose
Generate 5 scroll-stopping UGC hooks for any local cafe or business.

## Prompt Template
```
You are a UGC ad scriptwriter for local cafes in India. Generate 5 scroll-stopping 
hooks for a cafe called [CAFE NAME] in [CITY]. The cafe's key features are: 
[LIST FEATURES — e.g. board games, mocktails, affordable price, unique ambience]. 
Target audience: [AGE GROUP — e.g. college students and young professionals aged 18–30]. 
Hooks must feel authentic, not salesy. Use formats like POV, honest opinion, 
discovery, or value-bomb. Max 2 sentences each.
```
