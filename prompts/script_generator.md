# Prompt: Script Generator

## Purpose
Generate a full UGC-style ad script following the Problem → Solution → CTA framework.

## Prompt Template

```
Write a 30-second UGC-style Instagram Reels script for [CAFE NAME], [CITY]. 
Follow this structure:
- Hook (0–3s): Scroll-stopping opening line
- Problem (3–8s): A relatable frustration the audience has
- Discovery (8–18s): How they found this cafe and first impressions
- Experience (18–25s): Highlight 1–2 specific menu items or features with honest reactions
- CTA (25–30s): Soft, natural call-to-action — not salesy

Tone: real, conversational, like a friend recommending a place over chat.
Do NOT use promotional language like "amazing", "best ever", or "must visit".
Include natural hesitations and one honest observation (good or slightly critical).
```

## Variants

### 45-second YouTube Shorts
```
Write a 45-second UGC-style YouTube Shorts review script for [CAFE NAME], [CITY].
Structure: Hook (0–4s) → Setup/location context (4–12s) → The good stuff with 
specifics (12–28s) → One honest critique (28–38s) → Local CTA with engagement 
question (38–45s). Tone: honest reviewer, not influencer.
```

### 20-second Fast Visual Format
```
Write a 20-second text-on-screen fast-cut script for [CAFE NAME], [CITY].
Format: Each line is a separate on-screen text card with a 1-second visual cut.
Start with a question hook. Then list 4–5 features as short checkmarks. 
End with cafe name, location, and a one-line CTA. No voiceover needed.
```
