## 🤖 Reusable Prompt Templates

These prompts work for **any local cafe or business** — just fill in the brackets.

### Prompt 1 – Hook Generator
```
You are a UGC ad scriptwriter for local cafes in India. Generate 5 scroll-stopping 
hooks for a cafe called [CAFE NAME] in [CITY]. The cafe's key features are: 
[LIST FEATURES]. Target audience: [AGE GROUP]. Hooks must feel authentic, not salesy. 
Use formats like POV, honest opinion, discovery, or value-bomb. Max 2 sentences each.
```

### Prompt 2 – Full Script Generator
```
Write a 30-second UGC-style Instagram Reels script for [CAFE NAME], [CITY]. 
Follow this structure: Hook (0–3s) → Problem the audience relates to (3–8s) → 
Discovery of the cafe (8–18s) → Experience highlight with 1-2 specific menu items 
(18–25s) → Soft CTA (25–30s). Tone: real, conversational, like a friend recommending 
a place. Do NOT use promotional language. Include natural hesitations and honest 
observations.
```

### Prompt 3 – Caption + Hashtag Generator
```
Write 2 Instagram captions for [CAFE NAME] in [CITY] for a UGC-style post. 
Caption 1: Discovery tone (excited, personal, feels like a real visit). 
Caption 2: Honest review tone (balanced, authentic, mentions one thing that could 
be better). End each caption with 8–10 relevant local hashtags. Max 150 words per 
caption. Do not use the word "amazing" or generic phrases.
```
