# Script C – The Fast Visual Hook

**Platform:** Instagram Reels  
**Duration:** 20 seconds  
**Format:** Text-on-screen, fast cuts — no voiceover needed  
**Hook Type:** Value-bomb checklist  

---

## Script (Text Cards)

> Each line = one screen card, ~2-second cut

**Card 1:**  
"Looking for a cafe in Vizag that's actually different?"

**Card 2:**  
"Board games ✓"

**Card 3:**  
"Billiards table ✓"

**Card 4:**  
"Mocktails + coffee ✓"

**Card 5:**  
"₹450 for two ✓"

**Card 6:**  
"Superbikes outside ✓"

**Card 7 (Final frame):**  
"Area 51 – The Cafe Hideout  
Rushikonda, Vizag  
Your new hideout. 📍 Link in bio."

---

## Production Notes
- Use a trending upbeat audio — this format relies heavily on audio energy
- Font: bold, clean, white text on dark overlay or brand colors
- Each cut syncs to the beat of the audio
- B-roll per card: show the thing being mentioned (billiards table, mocktail, etc.)
- This format is ideal for paid Instagram ads — high thumb-stop rate
- No face needed — purely visual + text
